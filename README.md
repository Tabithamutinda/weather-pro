# WeatherPro 🌤️

A modern Android weather application built with Kotlin and Jetpack Compose that provides current weather conditions and hourly forecasts based on the user's location.

## Features

- Current weather conditions
- Hourly weather forecast
- Automatic location detection
- Sunrise & sunset times
- Weather caching
- Material 3 UI

## Tech Stack

- Kotlin
- Jetpack Compose
- Hilt
- Retrofit
- Coroutines
- StateFlow
- Navigation Compose

## Architecture

The project follows Clean Architecture with MVI

## Setup

1. Clone the repository
2. Add your API key in AuthInterceptor
3. Build and run the project
